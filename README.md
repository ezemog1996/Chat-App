### Description

This is a simple chat app using socket.io. Local storage is used to store the user, contacts, and conversations.

### Installation

go to https://amazingchatapp.herokuapp.com/

### Usage

It's use to chat with friends. Add friends using their id and you can create conversations with them

### Contributing

Contribute however you'd like

### Tests

Go to https://amazingchatapp.herokuapp.com/ and create a new id if you don't already have one. Add a contact using a new user's id then start a conversation with them. They should receive your messages and be able to reply.
